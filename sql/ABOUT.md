# 📝 Sobre o Projeto

## 🎯 Visão Geral

Este projeto de análise de dados SQL foi desenvolvido por **Diogo** com o objetivo de demonstrar habilidades avançadas em análise de dados, desde a estruturação do banco de dados até a criação de insights de negócio valiosos.

## 💡 Motivação

A análise de dados é fundamental para tomada de decisões estratégicas em qualquer negócio. Este projeto nasceu da paixão por transformar dados brutos em informações acionáveis que podem impulsionar o crescimento empresarial.

## 🛠️ Tecnologias Utilizadas

- **SQL**: Consultas complexas com window functions, CTEs e joins avançados
- **Banco de Dados**: SQLite (compatível com PostgreSQL e MySQL)
- **Análise**: RFM, Cohort, Market Basket Analysis
- **Documentação**: Markdown com explicações detalhadas

## 📊 Principais Análises

1. **Análise de Vendas**: Visão completa do desempenho comercial
2. **Segmentação de Clientes**: RFM para entender o valor do cliente
3. **Análise Temporal**: Sazonalidade e tendências
4. **KPIs**: Indicadores chave de performance com comparações
5. **Análise Avançada**: Retenção, churn e padrões de compra

## 🚀 Objetivos do Projeto

- ✅ Demonstrar proficiência em SQL avançado
- ✅ Criar análises práticas e aplicáveis ao mundo real
- ✅ Documentar cada passo para facilitar o aprendizado
- ✅ Oferecer um projeto completo para portfólio

## 🌟 Destaques

- **Código Limpo**: Consultas bem estruturadas e comentadas
- **Documentação Completa**: Cada análise explicada detalhadamente
- **Dados Realistas**: Exemplos que simulam cenários reais
- **Escalável**: Fácil de adaptar para diferentes negócios
